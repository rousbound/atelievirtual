Panorama

Dado es un paquete que ofrece algunas facilidades implementadas desde objetos de conexión de LuaSQL. Sus principales objetivos són:

  • mejores mensages en caso de errores,
  • iteradores para acceder a los resultados, y
  • más homogeneidad en algunas operacciones comunmente utilizadas en bancos de datos.

Dado es un software libre y usa la misma licencia de Lua 5.1.

Abstracciones sobre objetos

El paquete también contiene un módulo de objetos que provee un medio de definir clases y objetos que representen filas de tablas del banco con un esfuerzo mínimo. Dado no impone un sistema de clases, pero sugiere apenas un esquema con herencia simple:

  • objetos con recuperación automática de atributos del banco de datos
  • métodos diferentes para la creación de objetos y para la definición de clases

El mecanismo predeterminado usa clases para representar tablas del banco y instáncias para representar filas de tablas del banco. Sin embargo, es fácil definir atributos que no sean directamente relacionados con columnas de tablas, o clases que no sean relacionadas con tablas únicas.

Dado puede ser obtenida de su própia página.

Instalación

Dado es un conjunto de módulos para Lua 5.1 que deben ser instalados en su LUA_PATH, respetando la estructura de directorios de src del archivo de distribución. LuaRocks puede ser usado para instalar el paquete, ya que éste es también distribuido como un rock.

Manual

No existe un manual de referencia. La API es documentada a través de LuaDoc y está disponible apenas en inglés.
